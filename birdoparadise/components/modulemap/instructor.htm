Bird of Paradise

This component is an alternative way to view of modules.

Create an empty module in your course for each tab.
Use Stem to group the modules of your course into each tab.
You can access Stem in the left hand menu.
Modules in each group will display as clickable boxes that display assignments.
Students will see their progress as filled stars.

You can set the width of this component in the markup by wrapping it in a div.
div style="width: 790px; padding:5px;" component 'modulemap' /div
Scroll arrows are turned on if either the window or the parent of the component are smaller than the tab content.
