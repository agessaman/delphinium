Pop Quiz

How to create your Pop Quiz Game with this user friendly interface.

Select A Quiz
Start by selecting a quiz. When the next panel opens you will see the questions.

Select Quiz Questions
You can click, control+click or drag through the list to hilight questions.
Click the Add Selected Questions button to add them to your list of questions to use in a game.
If no questions are hilighted, the Add button will add all questions.

Questions Selected for Game
Modify the question list as needed, in this panel you can Remove Questions and See Quesiton Details
When your done click the Use These Questions button to save this list and preview them in the game.
You can select questions from multiple quizzes by choosing any quiz and adding more questions.
Students will receive points for playing the game. What a great way to learn.
If you have already chosen a group of questions and decide to change them, this panel will show first.
You can still remove or add questions to your liking, then click the Use button again.

Game Preview
Preview the game with the selected questions.
