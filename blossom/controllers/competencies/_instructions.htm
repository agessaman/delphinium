Competencies Component

Select the New Competencies Instance button.
Type a Name, select the Color for the amount, toggle the Animate switch and select a Size for the component.

 

Using the component:

In the CMS tab add a New Page and select the Blossom Competencies component.
Click the component to select the instance you want to use from the drop down list.

Setting up Competencies:

Using Stem create tags for assignments and quizzes.
Tags for competencies are designated using the prefix C: followed by a name.
Each name represents a group of assignments that belong to the competency.
When the component runs on a page it gathers all assignments and submissions, sorts them by the tag names and compares the amount completed to the total possible, then presents this as percent completed for each group.

 
  • The competency bar graph is divided by tenths and displays the percent earned.
  • Hovering on a bar displays points earned and points possible.
  • Clicking a bar displays the itemized assignments in the competency group.
  • The width of the component is 250px if the size is set to Medium.
  • The height is determined by the number of bars to display.
  • Each bar represents Assignments and Quizzes in a competency group.
  • Each graded item in a competency needs to have a tag starting with C:
  • Competencies are grouped by names. (C:Ideas, C:People, C:Align C:American History)
  • The name will be displayed above the bar.
  • Names can include spaces. (C:Computer Science)
  • Assignments can have other tags for other uses.
  • Assignments can belong to multiple competencies.
